# 🎵 Вероятностные автоматы: генератор музыки

**Демонстрационный проект по дисциплине «Модели и методы теории вычислений»**

Приложение наглядно показывает работу **Марковских цепей** (вероятностных автоматов) на примере генерации музыки в реальном времени.

---

### О проекте

Программа использует **вероятностный автомат** (Марковскую цепь) для генерации мелодий.  
Переходы между нотами не случайны: они учитывают музыкальную логику (близкие ноты в гамме имеют большую вероятность), при этом параметр **«Хаос»** позволяет регулировать степень случайности.

Поддерживаются:
- Два лада: **мажор** и **минор**
- Регулировка уровня хаоса (от полностью музыкальных переходов до полной случайности)
- Выбор инструмента: **Гитара** / **Пианино**
- Регулировка темпа (BPM)
- Отдельные слои: мелодия + бас + барабанный луп

---

### Возможности

- Генерация мелодий с помощью Марковской цепи
- Интерактивное управление параметрами в реальном времени
- Красивый полноэкранный интерфейс
- Два музыкальных инструмента с синтезированным звуком
- Полностью автономная работа (не требует интернета и дополнительных файлов)

---

### Технологии

- Python 3.11
- Pygame (интерфейс и звук)
- NumPy (генерация звуковых волн)
- Марковские цепи (вероятностные автоматы)

---

### Как запустить
Установите зависимости:
pip install pygame numpy

Запустите приложение:

python main.py
